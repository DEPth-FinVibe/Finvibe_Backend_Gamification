"학습 내용 미리보기"를 마크다운 리스트 형태로 작성하세요. 아래 조건을 모두 지키세요.

[출력 규칙]
- 반드시 한국어로만 작성
- 반드시 JSON만 반환: {"content": "..."}
- content 값은 마크다운 텍스트
- content 문자열 내부 줄바꿈은 반드시 \\n 으로 표현(중요)
- 불필요한 서론/홍보 문구는 줄이고, "무엇을 배우는지"가 바로 보이게 작성

[형식]
- 불릿 리스트(- ) 중심으로 4~8개 항목 생성
- 각 불릿은 학습자가 실제로 배우게 될 내용(개념/지표/사고법/체크포인트/실습)을 한 줄로 요약
- 너무 추상적인 표현("완벽 마스터", "확실한 수익") 금지
- 필요하면 마지막에 구분선(---) 1줄과 주의 문장 1줄을 추가할 수 있음

[예시(형식만 참고, 그대로 복사 금지)]
- XXX 기초 개념 이해하기\\n- 관련 지표/차트에서 무엇을 볼지 정리하기\\n- 자주 하는 오해와 주의점 짚기\\n- 실전 체크리스트로 적용해보기\\n\\n---\\n무리한 매매를 유도하지 않고, 학습 목적의 일반 정보만 다룹니다.
